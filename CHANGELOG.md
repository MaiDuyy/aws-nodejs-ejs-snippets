# Change Log

All notable changes to the "AWS Node.js & EJS Snippets" extension will be documented in this file.

## [1.0.0] - 2026-03-31

### Added
- **JavaScript Snippets:**
  - `ex-app` — Express App boilerplate với EJS view engine
  - `aws-config` — Cấu hình AWS SDK (DynamoDB + S3)
  - `aws-model` — CRUD operations cho DynamoDB model
  - `ex-router` — Express Router tích hợp Multer upload
  - `ex-ctrl` — Controller đầy đủ: upload S3, validation, tính toán giá, CRUD
- **EJS Snippets:**
  - `ejs-form` — Form template hỗ trợ upload file và hiển thị lỗi
  - `ejs-cards` — CSS Card Grid layout cho danh sách items
  - `ejs-search` — Thanh tìm kiếm và nút xóa tất cả
